Useful Information:

1. Using Git
http://www.railstutorial.org/book#sec:git_setup

OR

Short version:
1.1. git config --global user.name "Your Name"
1.1. git config --global user.email youremail@example.com

1.2. checkout project from my GitHub repository: 1.2.1. In the Git GUI, create repository with the project directory (empty project)
1.2.2. Setting GitHub (auth keys and repository in Git)
In Git GUI, Help > Show SSH key (that is the public key). If there is no key, you can generate one. This public key must be put in GitHub (administrative panel). Note: I think the key uses the e-mail set in Git.
1.2.3. Then Remote>Add, name=github, location=git@github.com:LightSoul/World-of-Sheldor.git
1.2.4. Remote > Fech From > github

2. (Re)Deployment on Heroku
2.1. Install => gem install heroku
2.2. Adding SSH keys (previously created for Git) => heroku keys:add
2.3. Pushing a site version => git push heroku [branch_name]
(this can be also done from GitGUI)
2.4. Migrating the database => heroku rake db:migrate
